- [x] El proyecto abre correctamente en VS Code.
- [x] `index.html` se ejecuta en el navegador.
- [x] La consola no muestra errores criticos.
- [x] El formulario valida campos vacios.
- [x] El formulario valida correo.
- [x] El formulario valida edad.
- [x] El formulario valida nota.
- [x] El formulario valida asistencia.
- [x] Los registros validos se agregan a la tabla.
- [x] El estado academico se calcula correctamente.
- [x] El boton limpiar funciona.
- [x] El boton cargar casos de prueba funciona.
- [x] El resumen se actualiza.
- [x] Complete la bitacora de depuracion.
- [x] Respondi el cuestionario.
- [x] Complete el mapa conceptual o notes.
