# 09 - Reflexion final

Responde con honestidad tecnica.

## 1. Que aprendiste esta semana?

Aprendi a revisar errores frecuentes en JavaScript de una forma mas ordenada. Tambien entendi mejor la importancia de leer la consola, comparar los ids del HTML con el codigo JavaScript y probar el sistema despues de cada correccion.


## 2. Cual fue el error mas importante que corregiste?

Uno de los errores mas importantes fue corregir la logica de validacion y clasificacion del estudiante. El sistema debia diferenciar entre Aprobado, Plan de mejora y No aprobado, por eso era necesario revisar bien las condiciones, los operadores y los datos numericos.


## 3. Como usaste la consola del navegador?

Use la consola para revisar si aparecian errores al cargar la pagina o al enviar el formulario. Tambien sirvio para confirmar que el archivo JavaScript ya no tenia errores criticos y que el sistema podia ejecutarse correctamente en el navegador.


## 4. Que estructura de control de flujo comprendiste mejor?

Comprendi mejor el uso de `if`, `else if` y `else`, porque esas estructuras permiten tomar decisiones segun la nota, la asistencia y los datos ingresados. Tambien use el ciclo `for` para recorrer los casos de prueba y agregarlos a la tabla.


## 5. Como aplicarias este aprendizaje en un proyecto real de tu comunidad o institucion?

Lo aplicaria en formularios para registrar estudiantes, actividades, asistencia o proyectos comunitarios. Antes de guardar los datos, validaria que la informacion este completa y correcta para evitar errores en reportes o decisiones academicas.


## 6. Que mejorarias si tuvieras mas tiempo?

Mejoraria la cantidad de pruebas manuales y agregaria mas evidencias visuales del proceso. Tambien podria organizar mejor los mensajes del sistema y probar mas casos invalidos para asegurar que todas las validaciones funcionen correctamente.
